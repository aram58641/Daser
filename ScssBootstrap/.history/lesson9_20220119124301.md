## Margin & Padding
  margin & Padding - ները դրանք լուսանցքներ են

  Bootstrap - ում դրանք գրվում են հետևյալ ձևով:

  Օրինակ 
  1. margine-top

```html
  <div class="mt-5">Welcome</div>
  
```
  2. margine-bottom

```html
  <div class="mb-5">Welcome</div>
  
```

  3. margine-left

```html
  <div class="ms-5">Welcome</div>
  
```

  4. margine-right

```html
  <div class="me-5">Welcome</div>
  
```
